Le projet

Cette application web a été réalisée dans le cadre du cours GTI-525 de l'École de Technologie Supérieure de Montréal lors de la session été 2021. Elle permet aux utilisateurs de visualiser de différentes manières les données météo historiques de différentes villes canadiennes.
